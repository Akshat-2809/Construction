# ACE Construction

A construction machinery rental/listing platform where users can browse machines, post their own equipment, send requests, and manage listings.

## Live Demo

https://construction-chi-six.vercel.app

## Tech Stack

**Frontend:** Next.js, React, TypeScript, Tailwind CSS, Firebase  
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, Cookies

## Features

- User authentication
- Browse construction machinery
- Add machine listings
- Manage personal listings
- Send machine rental requests
- Admin routes for platform management
- Contact, help, terms, and privacy pages
